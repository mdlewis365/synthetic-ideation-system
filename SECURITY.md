# Security Policy

## Scope

This repository contains documentation and sanitized examples only.

## Reporting Issues

Do not post private prompts, credentials, logs, generated private outputs, user data, or implementation details in public issues.

## What Is Not Included

The repository excludes source code, authentication logic, model-provider configuration, API keys, secrets, operational logs, raw invention requests, private prompt text, model-routing internals, and storage details.

## Safety Position

SIS output requires human review. Security-sensitive, safety-critical, regulated, patent-related, or engineering-dependent use cases require qualified review outside SIS.
