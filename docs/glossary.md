# Glossary

- Allowed Approaches: user-provided boundaries for acceptable concept directions.
- Causal Necessity Condition: optional control describing what must be causally necessary for a concept to count.
- Constraint-Inversion: invention mode that treats a constraint as the driver for ideation.
- Exploration Vector: the core direction of an SIS invention search.
- Invention Mode: user-selected mode that shapes the kind of concept output.
- Prior-Art Collapse Pattern: optional control describing obvious solution patterns to avoid.
- Reject If: user-defined rejection boundary.
- Scientist Input Module: structured SIS intake workflow.
- SIS: Synthetic Ideation System.

## What Is Not Included

The glossary does not expose private prompt labels, evaluator internals, source names, route names, scoring rules, or model-routing details.
