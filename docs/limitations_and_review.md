# Limitations and Review

## Limitations

SIS does not guarantee:

- novelty
- patentability
- freedom to operate
- physical feasibility
- safety
- engineering readiness
- regulatory compliance
- commercial viability
- experimental validation

## Required Review

Generated concepts should be reviewed by qualified humans. Patent-related use requires legal and prior-art review. Engineering or scientific claims require domain validation.

## Current Implementation Uncertainty

A separate invention-gate helper exists in the private codebase, but it was not found integrated into the active SIS submission path. Public SIS documentation therefore does not claim active automated gate enforcement beyond the confirmed input, mode, model, and private prompt-governance workflow.

## What Is Not Included

This document excludes private prompts, evaluator internals, gate rules, scoring thresholds, source code, logs, and raw user submissions.
