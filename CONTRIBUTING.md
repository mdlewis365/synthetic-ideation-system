# Contributing

This repository accepts public-safe documentation improvements only.

## Rules

- Do not add private SIS source code or prompt text.
- Do not publish private invention examples, user requests, evaluator details, scoring rules, model-routing logic, credentials, or logs.
- Do not claim active modules unless they are supported by the current implementation.
- Describe uncertain internals as conceptual or omit them.
- Preserve human-review and limitation language.

## What Is Not Included

Contributions must not include backend routes, private function names, storage schemas, raw output, operational artifacts, authentication details, or cloneable instructions.
